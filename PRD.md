# PRD ·「墨韵」— 网页水墨画绘制工具

> 版本：v1.0（已实现） · 最后更新：2026-06-11

## 一、产品定位

一款纯前端（HTML / CSS / JS，单页应用，无需服务器）的水墨画创作工具。以中国传统水墨美学为核心，追求「笔、墨、纸、水」四者的真实交融体验。克制、留白、有韵味。

**目标用户**：对中国画有兴趣的创作者、设计师、书法爱好者，以及希望体验水墨质感的普通用户。

**使用方式**：浏览器直接打开 `index.html`，无需安装、无需联网（字体 CDN 除外）。

---

## 二、核心功能

### 1. 画布与画廊

| 功能 | 说明 | 状态 |
|---|---|---|
| 新建画布 | 一键新建空白宣纸画布，默认以词牌名自动题名 | ✅ |
| 画廊收藏 | 保存画作至本地（localStorage，含 JPEG 缩略图），册页式排布展示 | ✅ |
| 画作管理 | 可题名、重新打开续画、删除；离开未保存时提示确认 | ✅ |
| 导出 | 将纸面 + 墨迹 + 流体层合成后导出为 PNG 下载 | ✅ |
| 涤纸 | 清空当前画布墨迹与流体状态 | ✅ |

画布尺寸：1000 × 1390 px（竖幅，接近传统立轴比例）。

### 2. 毛笔系统（4 种笔刷）

| 笔名 | 类型 | 特性 |
|---|---|---|
| 狼毫 · 工笔 | `fine` | 细劲挺拔，实线托底 + 戳印，线条连贯不断墨，湿墨注水减弱 |
| 羊毫 · 写意 | `soft` | 柔软饱满，随速度自然粗细变化，侧锋偏转明显 |
| 散锋 · 皴擦 | `bristle` | 多簇分叉笔毫，断续飞白，适合山石树皮皴擦肌理 |
| 斗笔 · 泼墨 | `broad` | 大笔阔墨，边缘自然洇散，抬笔不留出锋尾 |

**笔刷物理（已实现）**：

- 笔压模拟：基于鼠标速度——快则细枯、慢则粗润
- 起笔尖入：落笔以细锋起，随运笔渐展笔腹
- 侧锋：随机偏向一侧，墨心偏移，重边与轻边分明
- 收笔出锋：快速运笔抬笔后带出渐淡墨尾（工笔、斗笔除外）
- 运笔平滑：二次贝塞尔曲线圆化折线，指数平滑消除鼠标抖动
- 枯笔断墨：高干燥度时随机跳过戳印，呈现飞白

### 3. 墨的干湿度（8 级）

| 级别 | 透明度 | 干燥度 | 洇散 | 效果 |
|---|---|---|---|---|
| 焦墨 | 0.96 | 高 | 无 | 浓黑干涩，边缘锐利带飞白 |
| 浓墨 | 0.84 | 中 | 弱 | 黑实饱满，轻微晕渗 |
| 重墨 | 0.68 | 低 | 中弱 | 墨色厚实，开始有洇意 |
| 润墨 | 0.52 | 低 | 中 | 润泽饱满，边缘柔和扩散 |
| 淡墨 | 0.38 | 极低 | 强 | 半透明灰调，明显洇晕 |
| 轻墨 | 0.26 | 极低 | 较强 | 轻淡如烟，大范围渗化 |
| 清墨 | 0.16 | 无 | 很强 | 极淡，柔和渗化 |
| 烟墨 | 0.09 | 无 | 最强 | 几近透明，大面积晕染 |

湿度同时影响：墨色透明度、边缘扩散半径、流体网格注水量、与已有墨迹的混合程度。

### 4. 中国古典色系（8 色）

取自传统颜料与古典设色：

玄墨（松烟黑）· 黛青 · 花青 · 赭石 · 朱砂 · 藤黄 · 石绿 · 绛紫

颜色以「墨」的方式呈现——同样受干湿度控制，淡彩晕染如设色山水。各色预计算 RGB 与吸光系数，用于流体层减色混合。

### 5. 真实感渲染与水洗晕染

| 能力 | 说明 | 状态 |
|---|---|---|
| 多层合成 | 纸面 / 墨层 / 单笔缓冲 / 流体层 / 视图合成 | ✅ |
| 戳印笔刷 | 径向渐变墨体 + 枯笔颗粒噪点 | ✅ |
| 乘法混合 | 墨迹叠加自然加深，模拟积墨法 | ✅ |
| 流体渗流 | 细胞网格模拟湿墨持续向外洇散（每帧双步迭代） | ✅ |
| 纤维渗透率 | 双尺度噪声场 + 致密阻滞点，形成指状洇散边缘 | ✅ |
| 减色混合 | Beer-Lambert 吸光模型，多色颜料真实叠加 | ✅ |
| 清水笔 | 局部模糊 + 提墨 + 沿运笔方向推移 + 再润湿 | ✅ |

### 6. 纸张（4 种宣纸色调）

| 纸名 | 色调 | 效果 |
|---|---|---|
| 玉版宣 | 暖白微黄 | 经典生宣 |
| 仿古宣 | 米黄做旧 | 古旧感 |
| 茶笺 | 浅茶褐 | 暖褐底 |
| 月白笺 | 极淡青灰 | 清冷底 |

纸面生成：底色填充 + 3200 点肌理噪点 + 150 条纤维曲线 + 四边径向暗角。

### 7. 题款钤印

| 功能 | 说明 | 状态 |
|---|---|---|
| 题词 | 竖排自右而左，以空格分列，最多 40 字 | ✅ |
| 印文 | 一至四字朱文方印，边缘斑驳磨损 | ✅ |
| 落款 | 点选画面位置放置，钤印微斜 | ✅ |

### 8. 撤销

- 最多保留 6 步撤销（Ctrl + Z）
- 快照包含墨层 ImageData 与流体网格状态（水 + 三色颜料密度）
- 不含完整撤销树，保持克制

## 三、视觉与 UI 设计

### 整体审美

- 大面积留白，界面元素极少且收敛
- 工具栏如「文房陈设」般雅致，左侧竖排分组
- 朱砂红（`#9e3b26`）作为唯一强调色

### 字体

| 用途 | 字体 | 回退 |
|---|---|---|
| 标题 / 毛笔字 | Ma Shan Zheng（Google Fonts） | 楷体 |
| 正文 | Noto Serif SC | 宋体 |

### 中国元素

- 竖排品牌标题与印章式红色落款
- 画廊册页式卡片，悬停微动效如纸张轻浮
- 画布居中如悬挂宣纸，带极淡阴影
- 笔锋光标圆环跟随鼠标，清水笔时为虚线蓝圈

### 视图结构

1. **画廊视图**：画作网格 + 右侧竖排品牌区 + 底部留白脚注
2. **绘制视图**：左侧工具栏 + 中央画布 + 右上竖排题名

## 四、技术方案

| 项目 | 方案 |
|---|---|
| 架构 | 纯静态 SPA：`index.html` + `style.css` + `app.js` |
| 绘制引擎 | HTML5 Canvas 2D，多缓冲层 |
| 笔触 | Stamp-based 戳印 + 贝塞尔平滑 + 单笔缓冲提交 |
| 晕染 | 250×348 流体细胞网格，渗透率场驱动渗流 |
| 水洗 | 局部 blur 回写 + destination-out 提墨 + 网格再润湿 |
| 存储 | localStorage（`inkpaint.works`），含缩略图与墨层 PNG |
| 依赖 | 无 npm / 无构建；仅 Google Fonts CDN |

详见 [Architecture.md](./Architecture.md)。
